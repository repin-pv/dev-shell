## Для работы со строками

### ОбщийМодуль (Клиент)
```
Процедура ОбработкаСтрокТЧ(СтруктураДействий, СтрокаТЧ) Экспорт
	
Если СтруктураДействий.Свойство("РасчитатьОтколение") Тогда
		
		СтрокаТЧ.КоличествоОтклонение = - (СтрокаТЧ.КоличествоПоУчету - СтрокаТЧ.КоличествоПоФакту);
		
КонецЕсли;
	
Если СтруктураДействий.Свойство("ПересчитатьСумму") Тогда
		
		СтрокаТЧ.Сумма = СтрокаТЧ.Цена * СтрокаТЧ.Количество;
		
КонецЕсли;
	
КонецПроцедуры
```

### МодульФормы ПриИзменении
```
&НаКлиенте
Процедура НоменклатураКоличествоПриИзменении(Элемент)
	
	СтрокаТЧ = Элементы.Номенклатура.ТекущиеДанные;
	
	СтруктураДействий = Новый Структура;
	СтруктураДействий.Вставить("ПересчитатьСумму");	
	УО_РаботаСДокументами.ОбработкаСтрокТЧ(СтруктураДействий, СтрокаТЧ);
	
КонецПроцедуры

Процедура НоменклатураЦенаПриИзменении(Элемент)

	СтруктураДействий.Вставить("ПересчитатьСумму");	
	УО_РаботаСДокументами.ОбработкаСтрокТЧ(СтруктураДействий, СтрокаТЧ);
	
КонецПроцедуры
```
